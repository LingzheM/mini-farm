
## 🎯 第一部分：核心循环分析（Jesse的"透镜法"）

### 核心乐趣拆解

**你选的《星露谷物语》农场模块，有3层乐趣递进：**

1. **即时反馈** - 锄头挥下去，土地变了 ✓（你的prototype已有）
2. **计划性满足** - "我今天种的萝卜，3天后能收" ✗（缺时间推进）
3. **成长可视化** - 看着作物从种子→发芽→成熟 ✗（缺生长状态）

**Jesse会说：你的prototype处于Version 0.3阶段 - "能感受到操作"但"玩不了完整循环"**

## 🎨 第三部分：用户操作流程（Jesse强调的"玩家心流"）

```
【第一天 6:00】
玩家醒来 → 看到日期"春 1日" + 体力条100

↓ 拿起锄头
点击草地 → 播放挥动动画 → 土地变成耕地（棕色方块）
消耗10体力

↓ 拿起种子袋
点击耕地 → 播放播种动画 → 显示种子图标
消耗1个萝卜种子

↓ 拿起水壶
点击种子 → 播放浇水动画 → 土壤变深色 + 水滴特效
消耗10体力

↓ 体力用完
点击床铺 → 黑屏2秒 → 播放公鸡叫声


【第二天 6:00】
玩家醒来 → 看到"春 2日" + 体力恢复到100

↓ 自动检测
昨天浇水的种子 → 变成"发芽状态"（绿色小芽）
昨天没浇水的 → 还是种子状态

↓ 继续浇水...


【第四天 6:00】
萝卜进入"成熟状态"（红色萝卜图标）

↓ 收获
点击成熟作物 → 播放收获音效 → 作物消失 → 背包+1萝卜
显示"+1 萝卜"飘字
```

## ✅ 验收标准（Definition of Done）

### Sprint 1 完成标准

**必须能完成这个测试流程：**

1. ✅ 游戏开始，显示"春 1日 6:00"
2. ✅ 用锄头开垦3块地
3. ✅ 种下3颗萝卜种子
4. ✅ 给2颗浇水（故意漏掉1颗）
5. ✅ 点击床铺 → 黑屏 → 显示"春 2日 6:00"
6. ✅ 浇水的2颗变成"发芽状态"，未浇水的还是种子
7. ✅ 继续浇水并睡觉...
8. ✅ 第4天，成熟的萝卜可以点击收获
9. ✅ 背包显示"萝卜 x2"

**如果以上流程能跑通 → Sprint 1 成功！**
